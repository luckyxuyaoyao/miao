# todo-app
Vue+Webpack打造todo应用

此源码的Webpack版本为4.0.0以上，vue版本3.0及以上

有应用vue组件化的概念

要先安装好node环境

## Project setup

```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
localhost:8888
```

### Lints and fixes files
```
npm run build

或者：用yarn

yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
